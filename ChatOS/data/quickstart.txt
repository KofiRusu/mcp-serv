ChatOS Quick Start Guide

== Installation ==

1. Clone the repository:
   git clone https://github.com/KofiRusu/ChatOS-0.1.git
   cd ChatOS-0.1

2. Create a virtual environment:
   python -m venv .venv
   source .venv/bin/activate

3. Install dependencies:
   pip install -r ChatOS/requirements.txt

4. Run the server:
   uvicorn ChatOS.app:app --reload

5. Open http://localhost:8000 in your browser

== Using the Interface ==

- Type your message in the input box at the bottom
- Select "Normal" or "Code" mode from the dropdown
- Toggle RAG to include/exclude document context
- Click Send or press Enter
- The main response is shown as the assistant's reply
- Click "Show all responses" to see what each model said
- The winning model is highlighted

== Tips ==

- Use Code mode when asking programming questions
- Add your own documents to ChatOS/data/ for RAG
- Conversation memory keeps track of recent exchanges
- Try asking follow-up questions to test memory

== Keyboard Shortcuts ==

- Enter: Send message
- Shift+Enter: New line in message
- Escape: Clear input
